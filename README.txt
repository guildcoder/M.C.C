McNabb Cotton Club PWA

Files:
- index.html
- manifest.json
- service-worker.js
- assets/

How to use:
1. Upload this folder to GitHub Pages or any static web host.
2. Open on iPhone in Safari.
3. Use Share > Add to Home Screen.

Notes:
- Scores and player names save locally on the device.
- Built from the McNabbCottonClubScorecard-v2.pdf layout and content.
